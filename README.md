# :pencil2: 關於作品



這是一個參考UBER地圖搜尋及相關基礎互動功能所製作的clone app

內容涵蓋以下：

- 使用者資料的註冊與登入
- 使用者位置定位與周圍地點搜尋
- 乘客與司機個別化的特色標示、登入介面及功能操作
- 乘客/司機當前所在位置動態更新
- 目標地點路線規劃與標示
- 可個性化儲存常用位置(居住位置/工作地點)
